# 🧠 System Designer AI

> Transform your application ideas into comprehensive, production-ready system designs — powered by Google Gemini AI.

## Features

- **AI Architecture Analysis** — Paste any app idea and get a full system design
- **Live Architecture Diagrams** — Rendered with Mermaid.js (zoomable, pannable)
- **Implementation Roadmap** — Phased, actionable plans with complexity estimates
- **Technology Stack Recommendations** — Justified tech choices for your use case
- **Production Concerns** — Scalability, security, fault tolerance, caching, monitoring
- **Follow-up Chat** — Ask deeper questions after receiving your design
- **Design History** — All designs saved locally in your browser
- **Export** — Download designs as Markdown or copy to clipboard

## Setup

1. Get a free **Gemini API key** from [aistudio.google.com](https://aistudio.google.com/apikey)
2. Open `index.html` in your browser
3. Click ⚙️ Settings and enter your API key
4. Describe your app and hit **Generate Design**!

## Tech Stack

- **HTML/CSS/JS** — Pure vanilla, no framework
- **Google Gemini 2.0 Flash** — AI generation engine
- **Mermaid.js** — Architecture diagram rendering
- **Marked.js** — Markdown parsing

## Usage

Press `Ctrl+Enter` (or `Cmd+Enter`) in the textarea to quickly generate a design.
